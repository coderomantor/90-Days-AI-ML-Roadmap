# Day 13: Linear Regression Implementation

## Topic

Linear Regression Implementation

## What I Practiced

- Creating a house price dataset
- Separating features and target
- Splitting data into training and testing sets
- Training a `LinearRegression` model
- Making predictions
- Comparing actual and predicted values
- Evaluating predictions with MAE, MSE, RMSE, and R2 Score
- Interpreting intercept and coefficients
- Saving a prediction chart

## Project

**House Price Prediction (Part 2)**

This project implements a full beginner-friendly linear regression workflow using scikit-learn.

## Key Takeaway

Linear regression implementation connects theory to practice. A model is only useful when I can train it, make predictions, evaluate errors, and explain what the learned coefficients mean.
