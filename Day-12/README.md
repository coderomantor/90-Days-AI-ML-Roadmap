Linear Regression Theory

## What I Revised

- Features and target
- Linear regression formula

- Simple vs multiple linear regression

## Project

**House Price Prediction (Part 1)**

This project focuses on understanding the relationship between house area and price before training a model.

## Key Takeaway

Linear regression is used to predict numeric values by learning a relationship between input features and a target. Before implementing a model, it is important to understand what the model is trying to learn.
