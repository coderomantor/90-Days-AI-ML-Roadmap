## Practice Ideas

- Add a new categorical column and encode it.
- Create a new feature using two numeric columns.
- Compare label encoding, ordinal encoding, and one-hot encoding.
- Save the prepared dataset as a CSV file.
- Explain why each new feature could help a machine learning model.
