## Concepts Reviewed

### NumPy

NumPy helps with numerical data, arrays, vectorized calculations, and basic statistics.

Important ideas:

- Arrays
- Rows and columns
- Mean, max, min, and sum
- `axis=0` and `axis=1`

### Pandas

Pandas helps work with table-like data using DataFrames.

Important ideas:

- Creating DataFrames
- Selecting columns
- Creating new columns
- Checking data types
- Filtering and sorting data

### Data Cleaning

Data cleaning prepares raw data for analysis and machine learning.

Important ideas:

- Missing values
- Duplicate rows
- Data types
- Value counts
- GroupBy summaries

### Data Visualization

Visualization helps turn data into understandable patterns.

Important ideas:

- Bar charts
- Histograms
- Pie charts
- Boxplots
- Scatterplots
- Heatmaps

### Exploratory Data Analysis

EDA combines summaries, data quality checks, visualizations, and written insights.

Important ideas:

- `head()`
- `info()`
- `shape`
- `describe()`
- Missing value checks
- Duplicate checks
- Final insights
